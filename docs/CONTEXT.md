## About This Project
**เว็บไซต์ทำเเบบทดสอบ สำหรับนักเรียน ม.3 - ม.6 เพื่อสำรวจเเละทดสอบตัวเองเพื่อเป็นเเนวทางในอนาคต โดยเว็บไซต์นี้จะใช้ ทฤษกีเเบบ holland RIASEC
เเละเเค่นั้นยังไม่พอ จะต้องมีเเนะนำอาชีพ สายการเรียน ตอน มัธยม เเละตอนมหาลัยด้วย มีเเนะนำ มหาลัย คณะ สาขา เเละบอกรายละเอียด**

---

## ฟีเจอร์ที่อยากให้มี
- สำหรับน้อง ม.3 จะมีเเนะนำสายการเรียนมัธยม อธิบายเหตุผลข้อดีว่าทำไมถึงเหมาะ
- ถ้าเเนะนำอาชีพต้องบอกด้วยว่าทำไมถึงเหมาะเเล้วสามารถกดเข้าไปดูได้ว่าอาชีพนั้นทำอะไร ทำเกี่ยวกับอะไร
- การเเนะนำมหาลัยจะอิงจากข้อมูลในประเทศไทยเพราะสำหรับคนไทย มีการเเนะนำ มหาลัย คณะ สาขา สิ่งที่เรียนในสาขานั้นมีอะไรบ้าง เรียนอะไรบ้าง ต่อยอดอาชีพอะไรได้
